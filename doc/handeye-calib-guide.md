# 如何手眼标定

选取机械臂末端坐标系与相机安装外壳坐标系进行标定（眼在手上）。

- 为什么不选取相机光学坐标系？
  若选取光学坐标系作为标定目标坐标系，最终TF树中，光学坐标系会有两个父节点：机械臂末端坐标系与相机安装外壳坐标系，这违反了TF树中单一父节点的原则。

采用的标定板规格：

- 棋格板，型号GP290, 12x9的棋盘格，11x8的角点，单个格子20mm

标定准备：

1. 将相机固定在机械臂末端，确保相对机械臂末端不会有任何移动、旋转
2. 将标定板放置于相机基座约1米的地方，确保标定板在后续标定过程中相对机械臂基座不会有任何位姿变动
3. 配置通用机器人描述文件，确保相机安装在机械臂末端，并填入一些预估粗值

标定流程：

1. 启动机械臂、相机、标定程序
2. 调整机械臂末端位姿，确保末端停止运动时，相机画面可以清晰显示完整标定板
3. 在标定程序中交互窗口点击添加样本
4. 重复步骤2-3，尽可能多的覆盖不同情况：机械臂不同关节旋转角度，标定板在相机画面中的位置、倾斜角度，直到采集到15~30份样本
5. 完成标定，保存结果到文件
6. 将四元数格式的标定结果转为欧拉角格式，并写入通用机器人描述文件

关于标定程序：

通过相机画面求解PnP，得到的是标定板在相机光学坐标系下的位姿，而标定目标是机械臂末端坐标系->相机安装外壳坐标系，这时需要再通过相机内部的TF树，变换这个位姿到相机安装外壳坐标系，得到标定板在相机安装外壳坐标系下的位姿，进而用于求解方程 AX = XB。
